# Developer Portfolio

A simple developer portfolio website built with HTML, CSS, and JavaScript.

## Files

- `index.html` — main page structure
- `styles.css` — responsive styling
- `script.js` — mobile menu behavior

## Customize

1. Replace `Your Name` with your real name.
2. Update the project descriptions in the `Projects` section.
3. Change the contact email in `index.html`.
4. Add your own skills, experience, and links.

## Preview

Open `index.html` in a browser to view the portfolio.
